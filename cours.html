Créer un fichier HTML

Pour créer un fichier, il vous suffit de presser "ctrl + N" sur votre éditeur de code. Puis enregistrez le fichier en pressant "ctrl + S". Une fois ces touches pressé, votre explorateur de fichier qui va alors s'ouvrir.
Allez dans le dossier ou vous souhaitez placer votre fichier et nommez le comme vous voulez en terminant le nom par .html (exemple : "monsite.html")

Ouvrir un fichier HTML

Pour ouvrir un fichier, il vous suffit de presser "ctrl + O" sur votre éditeur de code. Une fois ces touches pressé, votre explorateur de fichier qui va alors s'ouvrir.
Allez dans le dossier ou vous souhaitez ouvrir votre fichier et cliquez 2 fois dessus.

Afficher vos modifications

Pour afficher vos modifications, vous devez enregistrer le fichier en pressant "ctrl + S" puis l'ouvrir sur votre explorateur de fichier avec un navigateur comme internet exploreur ou encore google chrome.

Si votre fichier est déjà ouvert sur votre navigateur, vous pouvez simplement actualiser la page en pressant "f5" ou "fn + f5"

Les types de liens !

Il y a plusieurs types de liens : les liens externes et les liens internes

Les liens externes sont des liens qui redirigent en dehors du site (exemple google)

Les liens internes sont des liens qui redirigent vers une autre page html de notre fichier
