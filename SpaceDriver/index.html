Space Driver


Welcome to Space Driver, a game where you have to survive from planet to planet to make the best score possible. Each planet corresponds to 1 mini game!

This project was carried out as part of a requested one button game project.

Try here


What I learnt


  • This game was the first to be made! The constraints were quite strong: we had to make a game that could only be played with one button. So I learnt how to manage a production schedule and create my first game from A to Z.
  • This game was also created for mobile phones. This constraint allowed me to learn how to optimize the game and adapt it to the needs of mobile phone users.
